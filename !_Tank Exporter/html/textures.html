Texture and UV Viewing

 



There is a button with some hills and a sun on the left panel. Clicking will show the buttons to select which textures to show.
Once this these buttons are visible, clicking one of them will show the buttons for each texture for that tank section.

Clicking one of the texture buttons selects that texture and opens the Texture Viewing Window.
In this window there are some buttons on the top.. They are pretty east to understand their functions.

If the UVs are set to be visible, mousing over the tank will show the polygon and its matching UV in the texture viewing window.

Pressing "C" will center any UV triangle in the texture viewing window.

You can save the image in its original size as a PNG. There is no reason to save the UV map in any other format.
Use this UV map as layer in your painting tool as a guide to where to paint.
